# NeuroDiag

Eine interaktive Webanwendung für umfassende psychologische und psychometrische Diagnostik zu Themen der Neurodivergenz.

## Überblick

NeuroDiag bietet Nutzenden die Möglichkeit, alle relevanten Diagnostik durchzuführen, analysiert und ausgewertet zu werden. Das Design ist professionell mit alternativen Indie-Nuancen, basierend auf einer fundierten Farbpalette und modernen UX-Prinzipien.

## Features

- **Diagnostik-Module**: AQ-Test (Autismus), ASRS-Test (ADHS), Dyslexie-Test, Dyspraxie-Test
- **Professionelles Design**: Sauberes, zugängliches Interface mit beruhigenden Farben
- **Datenschutz**: Fokussiert auf Sicherheit und ethische Standards

## Technologie

- PHP 8.5
- HTML5, CSS3, JavaScript
- Lokaler Entwicklungsserver

## Installation und Ausführung

1. PHP 8.5 installieren
2. Repository klonen
3. Server starten: `php -S localhost:8000`
4. Im Browser öffnen: http://localhost:8000
